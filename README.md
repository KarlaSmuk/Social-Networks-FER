# Social Networks Course

## Overview

This repository contains coursework and laboratory exercises for the Social Networks course at the Faculty of Electrical Engineering and Computing (FER), University of Zagreb.

## Technologies

- **Backend**: Java, Spring Boot
- **Frontend**: React
- **Database**: MongoDB
- **Data Sources**: Social media APIs

## Academic Context

This work is part of the academic curriculum at FER, focusing on practical implementation and analysis of social network concepts and algorithms.
